UI 问题分析

检查了 TVBox 应用的所有页面，发现以下主要问题：

1. 页面架构不一致

新设计系统 (Home, VodDetail, Player)

  • 使用统一的设计系统组件
  • 色彩: 琥珀色(#d89a57) + 蓝色(#75a9c3)
  • 组件类: .app-shell, .surface-panel, .media-card

旧页面 (Live, Vod, Settings)

  • 使用内联 Tailwind 类 (bg-gray-900)
  • 未使用设计系统组件
  • 视觉上与新页面不一致

2. 遗留页面问题

Live.vue 和 Vod.vue 作为独立页面存在，但它们的路由被重定向：

  • /live → 重定向到 /library/live
  • /vod → 重定向到 /library/movie

这些页面实际上是孤儿代码，只在直接访问时有用。

3. 全屏播放器问题

FullscreenPlayer.vue 存在但可能未被使用，因为 CSS 全屏已实现。

4. 命名不一致

  • 组件文件夹: detail/, home/, media/, player/ (小写)
  • 模板使用: <DetailHero>, <HomeHero> (PascalCase)
  • VodCard.vue 在根目录，但 MediaCard.vue 在 media/

选择一个修复优先级:

A

统一旧页面样式

将 Live.vue, Vod.vue, Settings.vue 重构使用设计系统

B

删除孤儿代码

移除 Live.vue, Vod.vue (路由已重定向) 和 FullscreenPlayer.vue (CSS 全屏已实现)

C

全部处理

先统一旧页面样式，再删除孤儿代码
